This is a simple web application developed using Python and Streamlit.  
The app allows users to enter their name and birthday and shows how many days are left until their next birthday.

## Features
- User-friendly interface
- Birthday countdown
- Celebration animations like balloons and snow
- Built using Python and Streamlit

## Technologies Used
- Python
- Streamlit
- Datetime

## How to run the project
1. Install Streamlit
2. Run the command:
   streamlit run birthday.py
